Responsive Preview

Iframes your local app (/, same origin) at each checklist breakpoint. Run npm start, then open http://localhost:4200/responsive-preview.html. Scroll horizontally to see all frames.

Manual checklist (verify visually in each frame)
  • 375px — no horizontal scroll · text readable · tap targets ≥44px · hamburger menu shows · images scale
  • 768px — tablet layout · 2-col grids · spacing balanced · nav clear
  • 1024px+ — full layout · 1200px max container · whitespace · desktop nav
